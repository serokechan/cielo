# Cielo — Safe Stablecoin Payment Copilot on Celo

## One-liner
AI copilot untuk bantu user melakukan pembayaran stablecoin di Celo dengan aman, jelas, dan tetap dalam kontrol user.

## Target tracks
- Best Agent on Celo
- Synthesis Open Track
- Agents that pay

## Problem
User baru bingung saat mau bayar pakai crypto:
- token mana
- network mana
- fee berapa
- salah kirim gimana
- agent boleh ngapain

## Solution
Agent membantu user:
- memahami intent pembayaran
- menyiapkan payment preview
- memberi warning/risk note
- meminta konfirmasi
- menjalankan atau mensimulasikan transaksi

## Final features
### Core
- Intent parsing
- Payment preview
- Risk / safety layer
- Human confirmation
- Execute / simulate transfer
- Receipt / activity log

### Nice to have
- Spending limit
- Recipient allowlist
- Recurring payment template
- Plain-language explanation
